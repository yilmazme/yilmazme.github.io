Hello, world!

My name is Mehmet. I am a programming enthusiast. My coding journey began with a search for Machine Learning. I am a physics graduate from METU. A term like Artificial Intelligence was very attractive to me and still is. My primary goal was learning some programming languages like Python and getting involved in Machine Learning with my physics background. Back then I was not so familiar with programming and did not know where to start. I began with HTML, CSS and continue with JavaScript. I was fascinated by the power of JavaScript on the web and coding with JavaScript began to be a hobby for me. Within JS' libraries or frameworks, my favorite is React. Using React, creating something with it is very amusing but, now I thinking about learning React Native and taking my skills set one level up by being able to create mobile apps. Besides still learning Python, C++ and some other JavaScript's libraries and frameworks. While coding I realize that very clearly: Just watching tutorials and try to learn all new technologies is not the best way to understand programming. That is why I start coding this site and the works you can see below.
